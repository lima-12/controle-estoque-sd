Como Usar

1. Importe o arquivo CSS no seu projeto:

<link rel="stylesheet" href="assets/css/botao.css">

2. Use as classes base:

<button class="btn-stok btn-stok-primary">Meu Botão</button>

3. Combine com outras classes para tamanhos e efeitos:

<button class="btn-stok btn-stok-primary btn-stok-lg btn-stok-pulse">Botão Grande com Pulse</button>
